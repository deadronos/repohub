# Candidates for deletion (or rewrite)

This file is a maintenance backlog for `docs/agents/`. It is not normative guidance.

## Current status

No immediate deletion candidates were identified during the 2026-03-22 docs alignment pass.

## What belongs here

Add entries only when a doc section is:

- redundant with a stronger source of truth
- historically interesting but no longer actionable
- vague enough that it causes drift or repeated interpretation errors

When adding an item, include:

- file path
- quoted text or section name
- why it should be removed or rewritten
- where the canonical guidance should live instead
